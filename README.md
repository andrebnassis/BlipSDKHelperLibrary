# Introduction 
BlipSDKHelperLibrary is a non-official library to help Blip users to create Messenger and BlipChat components.
There is a .NET Framework and a .NET Core version. See it below:

- [BlipSDKHelperLibrary .NET Core](https://github.com/andrebnassis/BlipSDKHelperLibrary.DotNet.Core)
- [BlipSDKHelperLibrary .NET Framework](https://github.com/andrebnassis/BlipSDKHelperLibrary.DotNet.Framework)

Feel free to contribute :)
